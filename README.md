Ecosistema digital de alto rendimiento para **Quad Rugby Chile**. Esta aplicación multi-página ha sido diseñada bajo el concepto de **Cinética Brutalista V2**, priorizando la claridad técnica, el impacto visual industrial y una arquitectura modular escalable para la comunidad de rugby en silla de ruedas en Chile.

## 🚀 Características Principales

- **Arquitectura Multi-página:** Navegación fluida entre módulos especializados (Reglas, Clasificación, Participación) mediante React Router.
- **Cinética Brutalista V2:** Estética premium con bordes duros, tipografía técnica (IBM Plex Mono, Space Grotesk) y contrastes extremos (#ff0000, #0057ff).
- **Manuales Técnicos Interactivos:**
  - **Reglamento:** Glosario táctico, comparativas de faltas y FAQ optimizado para SEO/AEO.
  - **Clasificación:** Guía detallada sobre la regla de los 8.0 puntos y perfiles funcionales.
- **Embudo de Captación Multi-rol:** Flujo de registro especializado para Atletas, Voluntarios, Árbitros y Clasificadores.
- **Formulario Dinámico de Contacto:** Sistema multi-paso condicional con validación en tiempo real y estados de éxito.
- **Rendimiento y Accesibilidad:** Optimización WPO y cumplimiento de estándares WCAG para una inclusión total.

## 🛠️ Tecnologías

- **Core:** React 18 + Vite 6
- **Enrutamiento:** React Router 7
- **Estilos:** Tailwind CSS 4 + Vanilla CSS (Design Tokens)
- **Animaciones:** Motion 12 (Framer Motion)
- **Iconografía:** Lucide Icons
- **Tipografía:** Epilogue, Space Grotesk, IBM Plex Mono (Google Fonts)

## 💻 Instalación y Uso

### Requisitos Previos

- Node.js 18.0 o superior
- npm o pnpm

### Pasos para iniciar el proyecto

1. **Clonar el repositorio**

   ```bash
   git clone [url-del-repositorio]
   ```

2. **Instalar dependencias**

   ```bash
   npm install
   ```

3. **Iniciar servidor de desarrollo**

   ```bash
   npm run dev
   ```

4. **Construir para producción**
   ```bash
   npm run build
   ```
