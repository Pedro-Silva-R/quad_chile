Una landing page de alto impacto visual diseñada bajo el concepto de **Cinética Brutalista** para la comunidad de Quad Rugby en Chile. Este proyecto combina un diseño crudo e industrial con micro-animaciones fluidas para una experiencia de usuario premium y moderna.

## 🚀 Características Principales

- **Estética Brutalista:** Uso de tipografías pesadas, altos contrastes y layouts asimétricos.
- **Componentes Dinámicos:** Secciones interactivas para Clubes, Actividad Reciente y Participación.
- **Diseño Responsive:** Totalmente optimizado para dispositivos móviles y escritorio.
- **Animaciones Premium:** Integración de `framer-motion` para transiciones suaves y efectos de scroll.
- **Dark/Light Mode:** Soporte nativo para temas visuales dinámicos.

## 🛠️ Tecnologías

- **Core:** React 18 + Vite 6
- **Estilos:** Tailwind CSS 4 (Beta)
- **UI:** Shadcn/ui + Lucide Icons
- **Animaciones:** Framer Motion (Motion 12)
- **Gráficos:** Recharts

## 📂 Estructura del Proyecto

```bash
src/
├── app/
│   └── components/     # Componentes modulares (Hero, Footer, Sections)
│       └── ui/         # Componentes base de Shadcn
├── styles/             # Configuración de temas y Tailwind
└── main.tsx            # Punto de entrada
```

## 💻 Instalación y Uso

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Generar bundle de producción
npm run build
```
